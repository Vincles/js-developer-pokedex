## 📝 Sobre o Projeto

Esta Pokédex é uma aplicação web que consome dados em tempo real da PokeAPI. Ela permite listar os 151 Pokémon da primeira geração, exibindo detalhes cruciais como tipos, atributos base (stats), habilidades e a cadeia completa de evoluções.

## ✨ Diferenciais Implementados:

Sistema de Abas: Navegação suave entre "About", "Stats" e "Evolution".

Conversão de Unidades: Cálculo automático de altura e peso (Métrico e Imperial).

Filtro de Lendários: Destaque visual exclusivo (Glow Effect) para Pokémon lendários e míticos.

Cadeia de Evolução: Lógica recursiva para buscar e exibir a árvore evolutiva de cada espécie.

Dark Mode: Interface adaptável para preferência de tema do usuário.

## 🛠️ Tecnologias Utilizadas

HTML5: Estruturação semântica.

CSS3: Layouts modernos (Flexbox/Grid), animações e variáveis para temas.

JavaScript (ES6+): Uso intenso de Async/Await, Fetch API, Promise.all e manipulação de arrays (map, filter, find).

IA: Para auxiliar na construção e desenvolvimento do projeto, também foi utilizado inteligência artificial para compreender melhor as linhas de código e utilizar a ferramenta para aprender o que cada função faz, não somente copiar. 

## 🎨 Inspiração

O design e a interface deste projeto foram inspirados no trabalho de Saepul Nahwan, adaptado para oferecer uma experiência de usuário fluida e nostálgica.
