# js-developer-pokedex
Pokédex interativa criada para consolidar conceitos de JS Assíncrono e Integração com APIs. Desenvolvida no curso da DIO (Prof. Renan Johannsen) com base no conceito visual de Saepul Nahwan. O projeto apresenta conversão de unidades (métrica/imperial), gráficos de base stats, cadeia de evoluções e interface responsiva com Dark Mode.
